# Web-Development_Project
Noir Brew is a visually engaging and user-friendly frontend website designed to bring the warmth of a cafe into the digital world. This project focuses on creating an immersive experience where users can explore, order, and connect with their favorite coffee blends effortlessly.
The platform allows users to order coffee online, browse a curated menu, and enjoy a seamless ordering experience. For those who prefer the café ambiance, it also provides a table booking feature, making it easy to reserve a spot in advance.
The website includes multiple sections such as a beautifully designed Home page, an informative About Us section, a functional Order page, and a secure Login system. Additionally, users can explore exclusive vouchers, premium offers, and special deals, enhancing the overall customer experience.
Designed with a modern and aesthetic interface, this project emphasizes smooth navigation, appealing layouts, and responsiveness. The goal was to simulate a real-world coffee brand website while strengthening frontend development skills.
